## Description  
  > For the flask, write a sample framework for generating a project.  

## Structure  
- <*project*>  
  - config.py  
  - run.py  
  - apps/  
    - \_\_init\_\_.py  
    - routes.py  
    - templates/  
      - base.html  
      - index.html  
      - sample.html  
    - static/  
      - default.css  

## Implementation  
    python.exe flask.py -h
